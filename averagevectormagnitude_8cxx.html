Detailed Description

Calculate the average magnitude of the vectors in a vector image. 

Usage:   pximagetovectorimage
     -in      inputFilenames, at least 2
     [-out]   outputFilename, default VECTOR.mhd
     [-s]     number of streams, default 1.
   Supported: 2D, 3D, (unsigned) char, (unsigned) short,   (unsigned) int, (unsigned) long, float, double.
   Note: make sure that the input images are of the same type, size, etc.
