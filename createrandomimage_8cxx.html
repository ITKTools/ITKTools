Detailed Description

Create a random image. 

This program creates a random image.

     Usage:
     pxcreaterandomimage
     	-out   	OutputImageFileName
     	-pt    	PixelType <SHORT, USHORT, INT, UINT, CHAR, UCHAR, FLOAT>
     	-id    	ImageDimension <2,3>
     	[-sd]  	SpaceDimension (the number of channels) <1,2,3>
     	-d0    	Size of dimension 0
     	-d1    	Size of dimension 1
     	[-d2]  	Size of dimension 2
     	[-r]   	The resolution of the random image <unsigned long>.
     			This determines the number of voxels set to a random value before blurring.
     			If set to 0, all voxels are set to a random value
     	[-sigma]	The standard deviation of the blurring filter
     	[-min] 	Minimum pixel value
     	[-max] 	Maximum pixel value
     	[-seed]	The random seed <int>
