Detailed Description

Perform T-test. 

Usage:   pxttest     -in      inputFilename     [-out]   output, choose one of {p,all}, default p                p: only print the p-value                all: print all     -c       the two data sample columns     [-tail]  one or two tailed, defauls = 2     [-type]  the type of the t-test, default = 1:                1: paired                2: two-sample equal variance                3: two-sample unequal variance     [-p]     the output precision, default = 8:   The input file should be in a certain format. No text is allowed.   No headers are allowed. The data samples should be displayed in columns.   Columns should be separated by a single space or tab.
