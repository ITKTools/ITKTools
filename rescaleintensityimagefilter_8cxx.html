Detailed Description

Rescale the intensities of an image. 

Usage:   pxrescaleintensityimagefilter     -in      inputFilename     [-out]   outputFilename, default in + INTENSITYRESCALED.mhd     [-mm]    minimum maximum, default: range of pixeltype     [-mv]    mean variance, default: 0.0 1.0     [-pt]    pixel type of input and output images;              default: automatically determined from the first input image.   Either "-mm" or "-mv" need to be specified.   Supported: 2D, 3D, (unsigned) short, (unsigned) char, float.   When applied to vector images, this program performs the operation on each channel separately.
