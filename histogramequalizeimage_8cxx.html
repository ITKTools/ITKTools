Detailed Description

Equalize the histogram of an image. 

This program applies histogram equalization to an image.
Works as described by Maintz, Introduction to Image Processing.
Usage:
pxhistogramequalizeimage
-in    	InputImageFileName
-out   	OutputImageFileName
-pt    	PixelType <FLOAT, SHORT, USHORT, INT, UINT, CHAR, UCHAR>
Currently only char, uchar, short, and ushort are supported.
-id    	ImageDimension <2,3>
