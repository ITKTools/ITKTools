Detailed Description

Create a grid image. 

Usage:   pxcreategridimage     [-in]    inputFilename, information about size, etc, is taken from it     -out     outputFilename     -sz      image size for each dimension     [-sp]    image spacing, default 1.0     -d       distance in pixels between two gridlines     [-stack] for 3D images, create a stack of 2D images, default false   Supported: 2D, 3D, short.
