Perform PCA. 

Usage:   pxpca     -in      inputFilenames     [-out]   outputDirectory, default equal to the inputFilename directory     [-opc]   the number of principal components that you want to output, default all     [-opct]  output pixel component type, default derived from the input image   Supported: 2D, 3D, (unsigned) char, (unsigned) short, (unsigned) int, (unsigned) long, float, double.
