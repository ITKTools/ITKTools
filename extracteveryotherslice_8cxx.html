Detailed Description

Extract every other slice of an image. 

Usage:   pxextracteveryotherslice     -in      inputFilename     [-out]   outputFilename, default in + EveryOtherKExtracted.mhd     [-K]     every other slice K, default 2     [-of]    offset, default 0     [-d]     direction, default is z-axes   Supported: 3D, (unsigned) char, (unsigned) short, float, double.
